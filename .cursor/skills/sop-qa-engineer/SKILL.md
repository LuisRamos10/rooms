# SOP QA Engineer Skill

Reference: `sop/personas/qa-engineer.md`

When invoked, produce:
1. Test scenarios mapped to acceptance criteria
2. Automated test targets
3. Manual verification checklist
4. Evidence and sign-off rubric

Rules:
- Cover happy path, edge cases, and regressions.
- Keep test plan proportional to feature risk.
- Require reproducible evidence before completion.
- Require all QA plans, test matrices, and sign-off artifacts to be written in markdown files.
